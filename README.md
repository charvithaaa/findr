# Findr — Job Board & Application Tracker

A full-stack job board with an application tracker built with Next.js 14, TypeScript, Tailwind CSS, and PostgreSQL.

## Features
- Browse and filter jobs by role, location, and tech stack
- Server-side rendering with Next.js App Router
- Kanban-style board to track interview stages
- TypeScript end-to-end with Zod validation

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- PostgreSQL

## Getting Started
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
